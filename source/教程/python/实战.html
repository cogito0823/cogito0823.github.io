虽然这个教程是面向小白的零基础Python教程，但是我们的目标不是学到60分，而是学到90分。

所以，用Python写一个真正的Web App吧！

  • python web 实践

    端庄的好喔 created at October 16, 2017 6:34 PM, Last updated at October 16, 2017 6:34 PM

    一年前跟着廖神的脚步，准备写一个个人博客，结果写着写着发现自己心血来潮停不下来，决定努力一把，看能不能写一个基于aiohttp的web框架，然后再基于这个框架实现自己的博客。经过了两个月的艰苦奋站，这个web框架算是有点样子。今天突然回忆起自己的python成长之路，想到可以把这个框架贴出来，大家相互指点学习一下。如果有兴趣的可以看一看。

    框架代码：框架代码 基于该框架实现的博客：博客代码

    虽然一年过去了自己还没有把这个博客上线。现在想一下主要有两点原因：

    1. 由于aiohttp只提供基础的异步request和response。很多web框架必须的东西都没有(比如，session, cache, locale,storage等)，需要自己来实现。最终导致整个框架的速度并没有想象中的那么快。
    2. 越写到后面越感觉没有信心，感觉市面上已经有那么多好而快的框架了，为什么要找虐自己去写一个呢（可能搞技术的都有这个情节吧，自己从零实现一个博客感觉很cool,事实证明并没有那么cool。现在终于领悟DRY原则的意思了）

    最终，感谢廖神能够写出这么好的python教程，谢谢你。
